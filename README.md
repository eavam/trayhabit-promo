# trayhabit-promo

Promo site for [TrayHabit](https://trayhabit.nomadixapps.org) — aligner wear time tracker for iPhone.

Plain static HTML/CSS, no build step. Deployed to GitHub Pages via Actions on every push to `main`.

## Pages

- `/` — landing (App Store marketing URL)
- `/privacy/` — Privacy Policy URL for App Store Connect
- `/terms/` — Terms of Use (linked from the app description)
- `/support/` — Support URL for App Store Connect
